# Summary

This module adds a content type filter to results from the core search 
module. The filter gives the user a quick way to run a second, more targeted 
search and uses AJAX to return the results.

For a full description of the module, visit the sandbox project page:
  http://drupal.org/sandbox/sarah/1186404
  
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/1186404

# Requirements 

AJAX module: http://drupal.org/project/ajax

# Installation

- Install as usual, see http://drupal.org/node/70151 for further information.
- Clear your theme cache, see http://drupal.org/node/337176 for further information.
